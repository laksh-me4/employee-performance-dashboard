# Employee Performance Management Dashboard (Angular)

A complete Angular project for tracking employee performance, reviews, goals, and department reports.

## Features
- Multi-page dashboard with Angular routing
- KPI overview with performance metrics
- Employee directory with search/filter and add employee form
- Review status management
- Goal progress tracker
- Department analytics reports
- Local storage persistence for updates

## Tech Stack
- Angular (standalone components)
- HTML + CSS + TypeScript (compiled to JavaScript)
- RxJS

## Project Structure
- `src/app/layout`: app shell + navigation
- `src/app/pages`: dashboard, employees, reviews, goals, reports pages
- `src/app/services`: shared data service
- `src/app/models`: domain models

## Run in Git Bash
1. Open Git Bash in this folder.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start development server:
   ```bash
   npm start
   ```
4. Open browser:
   ```
   http://localhost:4200
   ```

## Build
```bash
npm run build
```
